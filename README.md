# Devcontainer Configuration Template

This is a template for the `.devcontainer` directory used by [Dev Container](https://code.visualstudio.com/docs/devcontainers/containers).

## Usage

Copy this entire folder to your project root as `.devcontainer`:

```bash
cp -r /path/to/this/template /your/project/.devcontainer
```

Or use it as a reference to create your own configuration.
